How I Built a Simple Blog App with Vue 3 (and How You Can Too)

👋 Introduction

Published on April 20, 2025 · #project

I’ve always wanted a personal space to share my thoughts, research, and projects—without the clutter or limitations of large platforms. So I built a minimalist blog app using Vue 3, focused on simplicity, speed, and flexibility.

This post walks you through the entire process, from planning to deployment. You’ll find it especially helpful if you:

  • Prefer full control over your blog
  • Like lightweight setups with no CMS overhead
  • Want to learn Vue 3 through a hands-on project

📌 1. Project Plan and Goals

I started with a few key goals:

  • Keep the layout simple and mobile-friendly
  • Use plain files like .json and .html instead of a database
  • Avoid heavy dependencies like Tailwind CSS
  • Enable easy post creation via manual file editing
  • Make it open for future features like reactions, comments, and a dashboard

🧱 2. UI Mockup and Component Structure

Here's the basic layout I envisioned:

+--------------------------+
| Avatar + Name           |
| Menu: Projects | Posts  |
+--------------------------+
|     Post Cards Grid     |
|   (20 at a time + Load) |
+--------------------------+
| Scroll to Top Button    |
+--------------------------+

Main components:

  • Home.vue – Displays blog cards loaded from posts.json
  • PostPage.vue – Shows full post content from individual .html files
  • posts.json – Stores metadata like title, date, and filename
  • post-x.html – Actual blog post body (can include images, HTML, etc.)

⚙️ 3. Setting Up the Vue App

Here's how to create a new Vue 3 app using Vite:

npm create vite@latest vue-simple-blog --template vue
cd vue-simple-blog
npm install
npm run dev

Your folder will look like:

vue-simple-blog/
├── public/
│   └── post/
│       ├── post-1.html
│       └── post-2.html
├── src/
│   ├── components/
│   ├── pages/
│       ├── Home.vue
│       └── PostPage.vue
│   └── App.vue
├── posts.json

To enable routing between pages:

npm install vue-router

✍️ 4. Adding and Displaying Posts

To add a blog post:

  1. Create a new HTML file like post-3.html inside /public/post/
  2. Add a new entry in posts.json: 
    [
  {
    "id": 1,
    "title": "Welcome to My Blog",
    "date": "2025-05-07",
    "filename": "post-1.html"
  }
]
  3. The Home page loads this JSON and shows clickable cards
  4. Clicking a card navigates to PostPage.vue, which fetches and displays the HTML content
